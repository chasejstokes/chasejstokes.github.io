Part 1: HTML Basics

HTML stands for HyperTextMarkupLanguage. HTML defines thes tructure of a document.

HTML Syntax

HTML separates a web page into different elements. The elements are defined by HTML tags. Here is an example.

<html>
<body>
<p> Hi, I am a HTML element! </p>
</body>
</html>

Each tag has a start tag and a stop tag. For example, <p>...</p>. However, we can also use a shorthand for certain tags that do not require content, such as line breaks: <\br>

It is also conventional to indent the nested tags. This helps us to visualize the structure of the web page easily.
