Part 1: HTML Basics

HTML stands for HyperTextMarkupLanguage. HTML defines the structure of a document.

HTML Syntax

HTML separates a web page into different elements. The elements are defined by HTML tags. Here is an example.

<html>
<body>
<p> Hi, I am a HTML element! </p>
</body>
</html>

Each tag has a start tag and a stop tag. For example, <p>...</p>. However, we can also use a shorthand for certain tags that do not require content, such as line breaks: <\br>

It is also conventional to indent the nested tags. This helps us to visualize the structure of the web page easily.

There are many HTML tags that you can use. They can be found here. For more commonly used ones, you can check this out.

Practice

There are few ways you can write HTML in the tools we downloaded. It is easiest to write in VSC, which is what I would recommend for your final project.

However, writing HTML is also possible in Jupyter Notebooks.

    Pros
  • Easy to quickly update output without refreshing page
  • Creates a central location for turning in HTML code and Python
    Cons
  • No autofill of ending tags
  • No auto indent
  • Code highlights remain for Python, rather than for HTML

So, there are tradeoffs when writing HTML in Jupyter Notebooks. It'll be how you write for assignments, but I wouldn't recommend it for your final project.
