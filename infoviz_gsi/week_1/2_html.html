Part 1: HTML Basics

HTML stands for HyperTextMarkupLanguage. HTML defines the structure of a document.

HTML Syntax

HTML separates a web page into different elements. The elements are defined by HTML tags. Here is an example.

<html>
<body>
<p> Hi, I am a HTML element! </p>
</body>
</html>

Each tag has a start tag and a stop tag. For example, <p>...</p>. However, we can also use a shorthand for certain tags that do not require content, such as line breaks: <\br>

It is also conventional to indent the nested tags. This helps us to visualize the structure of the web page easily.

There are many HTML tags that you can use. They can be found here. For more commonly used ones, you can check this out.

Hyperlinks

Links are defined with the <a>...</a> tag.

Wikipedia, the Free Encyclopedia

<a href = "http://en.wikipedia.org" > Wikipedia, the Free Encyclopedia </a>

Images

Images are defined with the <img>...</img> tag. We have to specify the source (src) of the image as well. Here is an image of a cute cat.

<img src = "https://images.unsplash.com/photo-1607274641849-5a892286a7d3?ixlib=rb-4.0.3&ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&auto=format&fit=crop&w=1632&q=80" width = "70%" > </img>

Buttons

Buttons are defined using a button tag. Buttons are used for interaction.

<button > Click me to pet the cat </button>
